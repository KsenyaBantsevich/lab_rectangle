Была поставлена задача в описании класса прямоугольника со сторонами, параллельными осям координат.

Класс реализовывает следующие возможости:

1)Получение координат прямоугольника
2)Перемещение
3)Изменение размера
4)Увеличение размера на единицу по каждой из осей
5)Уменьшение размера на единицу по каждой из осей
6)Построение наименьшего прямоугольника, содержащего два заданных прямоугольника
7)Построение наименьшего прямоугольника, содержащего два заданных прямоугольника, с присвоением
8)Построение наибольшего прямоугольника, являющегося общей частью(пересечением) двух прямоугольников
9)Построение наибольшего прямоугольника, являющегося общей частью(пересечением) двух прямоугольников, с присвоением
