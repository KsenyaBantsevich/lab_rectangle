Поставленная задача: описать класс прямоугольника со сторонами, параллельными осям координат.

Класс реализовывает следующие возможости:

1) Получение координат прямоугольника
2) Перемещение
3) Изменение размера
4) Увеличение размера на единицу по каждой из осей
5) Уменьшение размера на единицу по каждой из осей
6) Построение наименьшего прямоугольника, содержащего два заданных прямоугольника
7) Построение наименьшего прямоугольника, содержащего два заданных прямоугольника, с присвоением
8) Построение наибольшего прямоугольника, являющегося общей частью(пересечением) двух прямоугольников
9) Построение наибольшего прямоугольника, являющегося общей частью(пересечением) двух прямоугольников, с присвоением

Методы класса: 
  1) void Input()- метод ввода с клавиатуры 
  2) void Result()- метод вывода на экран 
  3) void Peremeshenie()- метод для перемещения прямоугольника 
  4) void Size()- метод для изменения размера прямоугольника 
  5) void Size_na_1()- метод для изменения размера прямоугольника на 1 единицу 
  6) void Size_na_minus_1()-метод для изменения размера прямоугольника на 1 единицу 
  7) void Minimal()- метод для построения минимального прямоугольника 
  8) void MinimalSPrisv()- метод для построения минимального прямоугольника с присвоением 
  9) void Сross() - метод для построения пересечения 2 прямоугольников 
  10) void CrossSPrisv()- - метод для построения пересечения 2 прямоугольников, с присвоением 
  11) void Postroenie(int, int, int, int)- метод для рисования изображения прямоугольника
